# Frontend

React/Vite frontend for the Vehicle Service Inventory Management System.

## Run Locally

```powershell
cd frontend
npm install
npm run dev
```

The Vite dev server proxies `/api` requests to the Django backend at `http://127.0.0.1:8000`.

## Build

```powershell
npm run build
```
